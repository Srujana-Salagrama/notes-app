A simple and interactive **Notes App** built with **HTML, CSS, and JavaScript**. This project allows users to create, view, and delete notes easily.

## Features

- Add notes dynamically with a single click.
- Delete notes individually.
- Responsive design for desktop and mobile devices.
- Minimal and clean UI for ease of use.
- Each note is displayed as a box for better readability.

## Technologies Used

- **HTML** - Structure of the app
- **CSS** - Styling and responsive design
- **JavaScript** - Adding functionality and interactivity
